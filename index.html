FAQ

Questions answered.

Can Blobbed read my files?

No. Files are encrypted in your browser before upload. The decryption key never leaves your device. We physically cannot access your data.

What if Blobbed shuts down?

Your files live on Shelby Protocol nodes, not our servers. Even if this website disappears, anyone with your link can still download the file directly from the decentralized network.

Is there a file size limit?

Currently ~100MB per file for the MVP. Shelby Protocol supports much larger blobs — we'll raise limits as the network matures.

Why does upload require a wallet?

Registering a blob on Shelby requires an on-chain transaction on Aptos. This ensures ownership and provenance. Downloading is completely free and anonymous — no wallet needed.

How do I share a file?

After upload, click "Copy Link". The link contains the blob hash and encryption key in the URL fragment (after #). Send it via any channel — the recipient just opens the link and clicks download.
